Mission:
You are building the starting architecture for the DockWalk iOS repo.

DockWalk is an iOS-first Warehouse Management System for real dock workers, receivers, pickers, loaders, supervisors, small-to-mid-sized warehouses, and 3PLs that are priced out of heavyweight warehouse software.

This iOS repo owns:
- DockWalk iOS app
- SwiftUI navigation
- scanner-first WMS screens
- local/offline/sync scaffolding
- design system
- future camera evidence UX
- future PaymentManager seam
- App Store-ready product polish

This iOS repo does NOT own:
- DockWalk API implementation
- Supabase migrations
- Railway deployment
- WMS web/admin implementation
- SiteWalk code

Founder context:
Kyle worked in 3rd-party logistics for roughly 12 years before this. Treat DockWalk as a product shaped by real warehouse and dock experience. The first iOS client should feel sophisticated, beautifully simple, and built for long dock shifts.
